What is this?

This is a template editor for Minecraft Diamondfire, which is a server about coding minigames.
What this web application does is modify templates, which are items that save code. It aims to be easy, although it is in development, at its current stage, it can edit the raw data of a template enough to edit a template in any way.
I'm adding a visual programming method, similar to the discontinued DfVisual, to give an easy way of editing templates.

How to use it?

Importing your template.

Step 1: Get the code data. If you have Code Utilities, this is easy.
Hold your template, which you want to modify, and type /itemdata in chat. Then, click on any formatted, unformatted or /dfgive version. This should put some text on your clipboard.
Step 2: With this on your clipboard, you should paste it into the Code To Import box and press import. The website will automatically grab the template data and put it in the editor.

Editing Templates

There are multiple ways of doing this.
Method 1: editing the raw template data; if you understand JSON, you will edit it after looking at it for a bit. However, templates are formated a little strange.
Method 2: You can click on the previewed blocks, which shows you some info, like the text shown on the sign or all piston options. Click and edit is not fully released.
